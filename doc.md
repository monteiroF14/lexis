Por causa das limitações do projeto decidimos fazer uma aplicação misturada com um jogo dedicado à dislexia, tendo várias inspirações de outras aplicações como o Duolingo.

O Lexis vai ser uma aplicação onde pessoas com dislexia podem fazer fichas para treinar a sua fala através de pequenos exercícios desde combinação de letras para arranjar palavras, leitura de voz para a verificação da fala, entre outros. O público-alvo são as crianças e adolescentes com dislexia.

A aplicação vai se focar muito na gamificação para conseguir a retenção dos usuários. Essencialmente, irão existir dois modos de jogo: o modo normal - onde cada ficha dá XP para desbloquear a próxima; o modo "hardcore" onde o usuário ganha moedas para customizar o seu avatar.
